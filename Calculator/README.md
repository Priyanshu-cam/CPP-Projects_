 ## Working Explanation
- The program displays a menu and takes user input.
- Based on the selected option, the corresponding function of the Calculator class is executed.
- Every calculation is saved in a history vector.
- Users can view all previous calculations in a neatly formatted table.
- The program continues to run until the user chooses to exit.
