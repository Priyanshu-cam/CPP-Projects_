# Stack & Queue Implementation in C++

This project implements fundamental linear data structures
from scratch using C++.

## Stack Implementations
- Stack using Array
- Stack using Linked List
- Stack using Deque (STL)

## Queue Implementations
- Queue using Array
- Queue using Linked List
- Circular Queue

## Operations Supported
- Push / Pop (Stack)
- Enqueue / Dequeue (Queue)
- Peek / Front
- isEmpty
- Size

## Time Complexity
All operations:
- Push / Pop: O(1)
- Enqueue / Dequeue: O(1)

## Purpose
To understand internal working of stacks and queues,
memory handling, and performance trade-offs.
